What are the search functions?

Find All Drugs

Get a list of all drug products with the specified name. The name can be a drug ingredient, brand name, clinical dose form, branded dose form, clinical drug component, or branded drug component. For example, try one of these names: azithromycin, Lipitor, azithromycin 250 mg, morphine sulfate.

API Documentation
Approximate Match

Look for the drug name in the RxNorm data set that most closely matches the search. This function will display the name of the RxNorm drug, RxCUI, and score. If you are not sure the spelling of a drug, this is the function to use. For example, ranitidine (Zantac) is the correct spelling. Instead try: rantidine. You'll see even using incorrect spelling returns the proper drug name and RxCUI.

Image

What do medications and images have in common? Well, nothing. This is a fun addition to try out in between querying the NLM medication database. Simply, select Image from the dropdown menu, then search for the whatever you like! This function utilizes the microservice created by Alexander Shen.

What do all these terms mean?

  • API: Application Programming Interface. Simply, a way for us to connect to another computer and get information.
  • RxCUI: NLM describes it as "Identifier that points to the common meaning shared by the various source names grouped and assigned to a particular concept". In simpler terms, this is giving each medication an unique ID.
  • UUID: Universally unique identifier. Used for identifying information that needs to be unique within a system. This was generated by a microservice found here.
  • RxAUI: RxNorm atom unique identifier. This is a string identifier for each RxNorm concept.
  • Score: Found when using the Approximate Match search function. This is simply the match score (higher is better).
  • NDC: NLM describes it as "Identifier that points to the common meaning shared by the various source names grouped and assigned to a particular concept". In simpler terms, this is giving each medication an unique ID.
